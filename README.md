# 🚀 DevFolio AI Pro

DevFolio AI Pro is a modern and responsive Developer Portfolio Generator built using HTML, CSS, and JavaScript.

This project allows users to instantly create a beautiful developer portfolio preview by entering their details, skills, projects, and GitHub profile.

---

## ✨ Features

- 🎨 Modern Glassmorphism UI
- 🌗 Dark / Light Theme Toggle
- ⚡ Typing Animation
- 🖼️ Live Profile Image Upload
- 🏷️ Dynamic Skill Tags
- 📥 Download Portfolio as Image
- 📱 Fully Responsive Design
- 🔗 GitHub Profile Integration

---

## 🛠️ Technologies Used

- GitHub Pages
